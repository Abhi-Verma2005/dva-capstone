# DVA Capstone Project

This repository contains the dataset, documentation, and analysis for the DVA Capstone project.

## Folder Structure

- `dataset/`: Original datasets.
- `cleaned dataset/`: Cleaned and preprocessed data.
- `presentation/`: Project presentations and slides.
- `documentation/`: Project documentation and reports.
- `calculation and pivot table/`: Excel or other calculation files and pivot tables.
- `dashboards/`: Visualizations and dashboards.
